Block and Inline Elements

Block Elements

Block elements take up the full width available and always start on a new line.

Common Block Elements

  • <div> ... Generic container.
  • <p> ... Paragraph.
  • <h1> to <h6> ... Headings.
  • <ul> ... Unordered list.
  • <ol> ... Ordered list.
  • <li> ... List item.
  • <form> ... Form container.
  • <table> ... Table.
  • <section> ... Section of a document.
  • <header> ... Header of a page or section.
  • <footer> ... Footer of a page or section.
  • <nav> ... Navigation links.
  • <article> ... Self-contained content.
  • <aside> ... Side content.
  • <main> ... Main content.
  • <fieldset> ... Groups form controls.
  • <blockquote> ... Block quotation.
  • <pre> ... Preformatted text.
  • <canvas> ... Drawing area for graphics.
  • <dl> ... Description list.
  • <dt> ... Description term.
  • <dd> ... Description details.
  • <figure> ... Figure or image container.
  • <figcaption> ... Figure caption.
  • <address> ... Contact information.
  • <hr> ... Horizontal line (thematic break).
  • <tfoot> ... Table footer.

Inline Elements

Inline elements take up only the width they need and do not start on a new line.

Common Inline Elements

  • <a> ... Hyperlink.
  • <img> ... Image.
  • <span> ... Generic inline container.
  • <input> ... Input field.
  • <label> ... Label for an input.
  • <strong> ... Strong importance (bold by default).
  • <em> ... Emphasized text (italic by default).
  • <br> ... Line break.
  • <code> ... Code snippet.
  • <b> ... Bold text.
  • <i> ... Italic text.
  • <u> ... Underlined text.
  • <small> ... Smaller text.
  • <sub> ... Subscript.
  • <sup> ... Superscript.
  • <mark> ... Highlighted text.
  • <q> ... Short quotation.
  • <cite> ... Citation.
  • <kbd> ... Keyboard input.
  • <samp> ... Sample output.
  • <var> ... Variable.
  • <time> ... Time or date.
  • <abbr> ... Abbreviation.
  • <data> ... Machine-readable data.
  • <acronym> ... Acronym (Not supported in HTML5).

Difference

Block Elements Inline Elements
Start on a new line. Stay on the same line.
Take the full width available. Take only the required width.
Used for large sections of content. Used for small pieces of content or text.

Easy Way to Remember

Block = Starts on a new line and takes full width.

Inline = Stays on the same line and takes only the required width.
